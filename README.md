# ios_sample_proj

一個objective-c的範例程式碼

使用tdx open api來取得台北旅遊景點的相關資料

<br/>

包括了透過NSURLSession請求json與下載圖片

將json資料保存於可固化的model物件

使用非同步排程管理下載圖片，建立縮圖快取機制

最後呈現圖文資訊於自訂佈局的表單頁面，並能在滑動時快速的載入
